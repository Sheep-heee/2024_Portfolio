## 🌐 사용 기술

- **프론트엔드**: React, TypeScript, Tailwind CSS
- **애니메이션**: Framer Motion, CSS
- **상태관리**: Redux, React-Redux
- **버전 관리**: Git, GitHub
- **데이터베이스 및 호스팅**: Supabase
